# 💰 AI Money Mentor

AI-powered financial assistant for Indian youth 🇮🇳

---

## 🚀 Features

- 📊 Budget Planner  
- 💰 Investment Advisor  
- 💬 AI Chat Assistant  
- 📈 Dynamic Donut Chart (Investment Distribution)  
- 🧠 Smart Suggestions  
- 📊 Financial Health Indicator  
- 🎯 Goal Progress Tracker  
- 🕘 Chat History  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Groq API (LLM)  
- Matplotlib  

---

## 💡 Problem

Many young people in India struggle with:
- Managing expenses  
- Understanding investments  
- Planning financial goals  

---

## ✅ Solution

AI Money Mentor provides:
- Simple financial planning  
- Beginner-friendly investment advice  
- Real-time AI chat support  

---

## 📸 Features Overview

### 📊 Budget Planner
- Tracks income, expenses, savings  
- Shows financial health  
- Suggests smart plans  

### 💰 Investment Advisor
- Gives SIP-based investment plans  
- Shows visual chart (donut)  
- Easy to understand allocation  

### 💬 AI Chat
- Ask any financial question  
- Get instant AI responses  
- Maintains chat history

## ⚙️ Setup Instructions

1. Clone the repo  

```
git clone https://github.com/MethukuSrija/AI_Finance_Assistant.git
cd AI_Finance_Assistant
```

2. Install dependencies  

```
pip install -r requirements.txt
```

3. Add your Groq API Key  

```
client = Groq(api_key="YOUR_API_KEY")
```

4. Run app  

```
streamlit run app.py
```
